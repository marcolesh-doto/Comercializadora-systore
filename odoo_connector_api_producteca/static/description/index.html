Producteca Odoo Connector

Producteca Odoo Connector / Marketplace Connector for Odoo.

What it is?

Producteca Odoo Connector (odoo_connector_api_producteca) by Moldeo Interactive is a Marketplace Connector for Odoo.
This is the official Producteca Connector for Odoo developed by Moldeo Interactive and certified by Producteca.

Features

  • Post full products/prices/stock from Odoo to Producteca Marketplaces
    • Image links included
    • Variations with all attributes combinations
    • SKU and BARCODE options
    • Multiple locations
    • Multiple companies support
    • Multiple Producteca accounts by company
  • Get Orders, Clients and Shipping information from Producteca to Odoo
    • Import sale orders from every channel
    • Import clients biling info and delivery addresses
    • Use SKU or BARCODE to find existent Product
  • Notification and Invoices
    • Internal notifications when Producteca list Stock and Prices from Odoo
    • Realtime notification for each order
    • Invoices are created and sent to Producteca
